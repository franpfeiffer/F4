# F4

## Requirements

- Rust (stable)

## Build

```bash
cargo build --release
```

The binary will be at `target/release/f4`.

## Run

```bash
cargo run
```

Or after building:

```bash
./target/release/f4
```

## Install

```bash
cargo install --path .
```

### Install on Windows:

Download the executable on the last [release](https://github.com/franpfeiffer/F4/releases).
